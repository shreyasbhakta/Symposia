# React Main UI – WordPress theme

Copy the **`react-main-ui`** folder to your WordPress **`wp-content/themes/`** directory. Then:

1. Build the React app: `npm run build` (in the main project root).
2. Copy the contents of **`dist/assets/`** into **`react-main-ui/assets/`** on the server.
3. In WordPress admin, go to **Appearance → Themes** and activate **React Main UI**.

Full steps and troubleshooting: **[../docs/DEPLOYMENT.md](../docs/DEPLOYMENT.md)**
